# Foundation

Spectre is an AI-first development environment that lets skilled builders scale themselves across multiple complex projects simultaneously. This section captures the understanding behind what we're building — the purpose, the principles that guide decisions, and the boundaries that keep us focused.

Read Foundation before diving into the Codebase. This context ensures changes align with what we're trying to build, not just how it's currently implemented.

  Treat documentation as Spectre's durable project memory, not as a secondary artifact. Specs, plans, implementation, and docs updates are valuable because they preserve why the system exists, how it works, and how it changed over time.
</Decision>
