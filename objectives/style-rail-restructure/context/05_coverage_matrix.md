# StyleRail Theming Coverage

The audit covers 27 surfaces: 16 document block types, five non-block theme surfaces, and six Layout panes. Each row compares the registry, rendered controls, default theme file, and live CSS-variable consumers. A row is closed only when those paths agree or when a shared section knob owns the consumed variable.

## Result

- Registry: 21 entries after adding `sequence`; 16 block files and five non-block files.
- Default theme: 21 matching files in `themes/default/components/`.
- Matrix: 9 closed, 17 open, and 1 waiver requested.
- Dead registry token: `interaction-surface.noteFg`; its variable is present in an unused class constant.
- Missing theme files: none.
